Pacman Recreation

May 2023 - Jun 2023

This was a final high school project where I worked with a partner. I developed and worked on the visual and front-end components of this Pacman recreation.

GitHub Repository

Role and Contributions

My primary role was to develop the front-end components of the game. I was responsible for designing the user interface, creating visual assets in Photoshop, and implementing the game visuals using Java Swing. Additionally, I assisted with some back-end issues, such as making Pacman move seamlessly from one side of the screen to the other.

Challenges and Solutions

One of the main challenges was figuring out how to update the screen efficiently. Initially, the game had performance issues, but we optimized the rendering process to ensure smooth gameplay. Another challenge was implementing the wrap-around movement for Pacman, which required careful handling of game state and screen boundaries. Through collaboration and iterative testing, we successfully resolved these issues.

Learnings and Outcomes

This project taught me how to work effectively with Java Swing and JFrame for creating interactive applications. I also learned the importance of teamwork and communication, as collaborating with my partner was crucial for overcoming challenges. Additionally, I gained experience in optimizing game performance and handling complex game mechanics.
