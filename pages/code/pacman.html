Pac-Man Recreation

Timeline: May 2023 - June 2023
Java Game Development Algorithm OOP

A Java implementation of the classic Pac-Man arcade game featuring custom graphics, intelligent ghost AI behavior using pathfinding algorithms, and authentic gameplay mechanics. The project demonstrates object-oriented programming principles and game development concepts.

Project Overview

This project is a recreation of the iconic Pac-Man arcade game, developed using Java. The game features all the classic elements: a maze, Pac-Man character, ghosts with distinct behaviors, pellets, power pellets, and fruit bonuses. The implementation focuses on reproducing the original gameplay experience while demonstrating object-oriented programming principles and game development concepts.

The game includes a scoring system, multiple levels with increasing difficulty, and sound effects that enhance the gaming experience. The ghost AI implements custom pathfinding algorithms to create challenging gameplay that mimics the behavior of the original arcade ghosts.

Role and Contributions

As the developer of this project, my responsibilities included:

  • Designing the game architecture using object-oriented principles
  • Implementing the core game mechanics and controls
  • Creating the maze generation and rendering system
  • Developing intelligent ghost AI with different behavior patterns
  • Implementing collision detection and game state management
  • Designing and integrating the scoring system
  • Creating the game's visual assets and UI

Challenges and Solutions

Ghost AI Behavior

Challenge: Implementing distinct personalities and behaviors for each ghost that mimicked the original game.

Solution: Developed a custom pathfinding algorithm with different targeting strategies for each ghost. Blinky directly targets Pac-Man, Pinky attempts to ambush, Inky uses a combined approach, and Clyde alternates between chasing and retreating.

Collision Detection

Challenge: Creating efficient collision detection in a maze environment.

Solution: Implemented a grid-based collision system that optimized performance by only checking collisions in the immediate vicinity of game entities. This approach ensured smooth gameplay even when multiple entities were moving simultaneously.

Game State Management

Challenge: Managing different game states (start screen, gameplay, level transitions, game over) seamlessly.

Solution: Created a state machine pattern that handled transitions between different game states, ensuring proper resource management and user experience continuity throughout the gameplay session.

Technical Implementation

The game was developed using the following technologies and approaches:

  • Java: Core programming language with Swing for the graphical user interface
  • Object-Oriented Design: Used inheritance and composition to create a flexible game entity system
  • Custom Game Loop: Implemented a fixed-time step game loop for consistent gameplay across different hardware
  • Pathfinding: Created modified A* and BFS algorithms for ghost movement and targeting
  • Sprite Animation: Designed a simple sprite animation system for character movements and effects

The project was completed successfully with all planned features implemented, resulting in a fully playable Pac-Man game that remains faithful to the original arcade experience while demonstrating modern programming techniques.
