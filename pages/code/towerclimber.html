Tower Climber

Jun 2022 - Jul 2022

This is a JFrame game I made as a final project in grade 11, a math RPG turn-based game.

GitHub Repository

Role and Contributions

I developed the entirety of this webpage. Implementing the webpage using HTML, CSS, and JavaScript. Additionally, I handled various functionalities such as flexbox layouts, overlays, transitions, and cookies with JavaScript.

Challenges and Solutions

One of the main challenges was figuring out how to use flexbox for layout and handling overlays and transitions. Initially, the webpage had some layout issues, but I optimized the CSS to ensure a responsive design. Another challenge was implementing cookies with JavaScript, which required careful handling of user preferences. Through iterative testing and debugging, I successfully resolved these issues.

Learnings and Outcomes

This project taught me how to work effectively with HTML, CSS, and JavaScript for creating interactive and responsive webpages. I also learned the importance of debugging and testing, as well as handling user preferences with cookies. Additionally, I gained experience in optimizing webpage performance and handling complex layouts.

Game Development

I developed the entire code and game. It's coded completely in Java using a Java Swing JFrame as well as all its visuals. Some challenges were trying to figure out how to properly keep track of player data within the game, which could have been solved by a double array, which I wasn't aware of at the time. It taught me a lot about Java and how to develop a game using Swing and Java in general.
